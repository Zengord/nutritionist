# Nutritionist Landing Page

Лендинг нутрициолога с адаптивной версткой, слайдером, аккордеоном, попапом и анимацией при скролле.

Проект выполнен по готовому макету для практики верстки лендингов, адаптива и JavaScript-интерактива.

## Live Demo

https://zengord.github.io/nutritionist/

## Возможности

- Адаптивная верстка до 320px
- Слайдер
- Аккордеон
- Попап
- Анимация при скролле
- Плавный скролл при клике на пункт меню
- Использование rem и em единиц

## Стек

- HTML5
- CSS3 / SCSS
- JavaScript
- BEM
- Gulp/Webpack
- GitHub Pages

## Что показывает проект

Проект демонстрирует навыки верстки лендингов, работы с адаптивом, интерактивными элементами и анимацией интерфейса.
